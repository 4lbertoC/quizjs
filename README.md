> A server for QuizJs


## Install

```sh
$ npm install --save quizjs-server
```


## Usage

```js
var quizjsServer = require('quizjs-server');

quizjsServer('Rainbow');
```

```sh
$ npm install --global quizjs-server
$ quizjs-server --help
```
